# Submission Guide

1. Create an empty Github Repo for this lab (you could create a single repository for all labs)
1. Add the following .gitignore file at the root of your repository (make sure it is named as `.gitignore`):
[.gitignore](https://github.com/github/gitignore/blob/main/Unity.gitignore)
1. Add all your project files (Source code) to your Github repo. 
1. If you get errors about large file sizes, it means the `.gitignore` file was not added properly so go back to step 2.
1. If you still get errors about large file sizes, you may consult the following [lfs guide](./github_lfs.md) on how to set up Github LFS Archives for those large files.
