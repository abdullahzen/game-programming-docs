# Task 1
* Look for a red fire, a blue fire, and a smoke pngs on the pngitem.com from the references.
* Add the following features to the game:
  * Add two fire effects one blue, and one red.
    - Blue fire (on the right of the lane): when the ball passes through it, the game ends.
    - Red fire (on the left of the lane): when the ball passes through it, print a health decrease message (from 100 to 0) in the debug logs. Note that this behaviour should keep happening as long as the ball is still inside the fire.
  
* Look for a burning sound file, and add it to the red fire to only play while the ball is inside the fire.
