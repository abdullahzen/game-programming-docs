# Lab 6 - A

## Prerequisites
* `git clone https://github.com/abdullahzen/game-programming-labs`
* Copy the folder `Lab6A` into another location (where you can save it for your own reference)
* Load the project in unity

# Task 1
* Create a main menu with the following:
    * Start Game: ball should be hidden and only spawns after start game is pressed
    * Options: pops up another menu which contains:
        * A selector for preset colors for the ball that will spawn. Based on this selection, the ball spawns with the selected color.
        * An input field to specify the starting mass of the ball.
    * Quit: exits the game
* Game should pause when Esc is pressed on the keyboard and the following menu should appear:
    * Resume Game: resumes the game.
    * Main Menu: restarts the game.

# Task 2
* Implement a simple scoring system and display the score at the bottom right corner of the screen:
    * The score for the bowling game should reflect the number of pins that fell down.
    * If the throw is not a strike (not all the pins go down), a new ball should respawn.

# Task 3
* Implement a nice UI layout to show a victory sign of your choice at the end of the game when all the pins are down.
