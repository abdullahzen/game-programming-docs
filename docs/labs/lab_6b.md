# Task 1
* Create a main menu with the following:
    * Start Game: hero should be hidden and only spawns after start game is pressed
    * Options: pops up another menu which contains:
        * An input field to specify the enemies health (should modify the default value used by the enemy script). Enemy health is a field that indicates how many hits it will take for the enemy to be killed.
    * Quit: exits the game
* Game should pause when Esc is pressed on the keyboard and the following menu should appear:
    * Resume Game: resumes the game.
    * Main Menu: restarts the game.

# Task 2
* Implement a simple scoring system and display the score with the number of hits on the bottom right corner of the screen:
    * The score for the game should reflect the number of decoys killed.
    * Both the number of hits and score are cumulative throughout the game.

# Task 3
* Implement a nice UI layout to show a victory sign of your choice at the end of the game when all decoys are dead.
